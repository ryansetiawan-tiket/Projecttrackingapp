### Legend:
- `▼` = Expanded folder
- `▶` = Collapsed folder  
- `[🎯]` = Assign to Asset button
- `[Batch🎯]` = Batch assign all children to asset
- `[×]` = Remove item
- `[✏️]` = Edit inline (click to activate)
- `✅` = Valid input
- `⚠️` = Validation error (red border)

## 🎨 Color & Styling Guide

### Validation States
- **Valid input:** Green border `border-green-500`
- **Invalid input:** Red border `border-red-500`
- **Warning:** Amber border `border-amber-500`

### Icons
- Folder: `📁` or `<FolderIcon />`
- File: `🖼️` or `<ImageIcon />`
- Assign: `🎯` or `<Target />`
- Edit: `✏️` or `<Pencil />`
- Remove: `×` or `<X />`
- Success: `✅` or `<Check />`
- Error: `❌` or `<XCircle />`
- Warning: `⚠️` or `<AlertTriangle />`

### Spacing
- Dialog padding: `p-6`
- Section spacing: `space-y-4`
- Input gap: `gap-2`
- Tree indent: `ml-6` per level
